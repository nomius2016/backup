珠仔

即选号玩法。珠仔玩法是在80个号码中选出1至5个号码组合成一组进行的投注。玩家将选投注号码与中奖号码对照，不需按照顺序，根据所选号码与中奖号码相符的个数多少，来确定相应的中奖级别。分为正赌、反赌、全中三种下法，并且其中只有”正赌”玩法，有提供单一投注多重中奖机会，见下例。

  1. 正赌、全中

    2. 例：投注者购买的是08，18，38，58这四个号码为一个组合，且该期开奖号码中包含08，18，38，58这四个数字，则视为投注"4中4"，若开出号码中只有18，38，58则视为投注"4中3"，赔率以下表派彩。

    正赌赔率设置

    选5 选4 选3 选2 选1
    中5 251 - - - -
    中4 21 51 - - -
    中3 3 6 20.3 - -
    中2 0 2 2.3 10.3 -
    中1 0 0 0 0 3.3

    中5 1150 - - - -
    中4 0 240 - - -
    中3 0 0 48 - -
    中2 0 0 0 12 -
    中1 0 0 0 0 0
  2. 反赌

    3. 例：投注者购买的是02，04，06，08，10，12，14，16，18，20，22，24这12个号码为一个组合，且该期开奖号码中全不包含，则视为赢。
